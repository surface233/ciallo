## 🚀 一键部署（GitHub Pages）

本项目是纯静态页面，**无需服务器，无需构建**，可直接部署到 GitHub Pages：

1. **Fork 本仓库**（或直接 clone）
2. 进入仓库 **Settings → Pages**
3. 在 **“Source”** 处选择 **“Deploy from a branch”**
4. **Branch** 选择 `main`，文件夹选择 `/(root)`
5. 点击 **Save**
6. 等待 1~2 分钟，你的站点就会出现在  
   `https://你的用户名.github.io/仓库名/`

> 如果你有自己的域名，可以在 Pages 设置中的 **“Custom domain”** 处绑定，并去 域名提供商 添加 CNAME 记录指向 `你的用户名.github.io`。

## 🎮 如何游玩

1. 打开页面，好感度初始为 50
2. 跟随剧情选择不同的互动选项
3. 好感度会影响结局走向
4. 每步都有对应的图片和语音反馈
5. 点击 **「点击 Ciallo~」** 按钮有惊喜语音彩蛋 ✨
